# Rock Paper Scissors: Evolution Edition 🚀

A high-fidelity "Developer Dark Mode" Rock Paper Scissors application that bridges the gap between a classic C++ console implementation and a modern, glassmorphic web experience.

## 🌟 The Evolution
This project showcases the journey of a simple game logic from a **C++ CLI tool** to a **Premium Web Application**.

### 💻 Web Version (Modern)
A stunning, responsive web interface designed for developers.
- **Theme**: IDE-inspired "Deep Navy" Dark Mode.
- **Aesthetics**: Glassmorphism, neon blue accents, and smooth `cubic-bezier` transitions.
- **Realistic Icons**: High-fidelity, anatomically accurate SVG paths for Rock (stone), Paper (document), and Scissors (tool).
- **Neural Scan Logic**: A 2.5-second "Thinking State" featuring a scanning line animation and a processing progress bar to enhance suspense.
- **Responsive**: Mobile-first design using `clamp()` typography and flexible grid layouts.

### 📜 C++ Version (Legacy)
The original logic foundation that started it all.
- Demonstrates basic input handling and conditional logic.
- Implements random number generation with `rand()` and `srand(time(0))`.
- Pure CLI experience for performance and simplicity.

---

## 🎮 How to Play

### Web Interface
1. Open `index.html` in any modern browser.
2. Select your tool (Rock, Paper, or Scissors) using the interactive glassmorphic buttons.
3. Wait for the **Server neural scan** to complete.
4. View the result and check the persistent scoreboard.

### C++ Console
1. Compile the code using `g++ rps_game.cpp -o rps`.
2. Run the executable: `./rps`.
3. Enter `1`, `2`, or `3` when prompted.
